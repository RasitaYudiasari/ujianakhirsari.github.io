About Me

Nama saya Rasita Yudiasari, saya lahir di kubu raya 10 juli 2002. Dan saya tinggal di desa mega timur, kubu raya, kalimantan barat. jenis kelamin saya perempuan. saya beragama islam.

Kegiatan saya saat ini adalah Sebagai Mahasiswa di Sekolah Tinggi Ilmu Ekonomi Indonesia Pontianak ( STIE Indonesia Pontianak ) dengan program studi S1 Manajemen.

Saya anak kedua dari dua bersaudara. Ayah saya bernama sunandi dan ibu saya bernama Mulyana.

Riwayat pendidikan :

- SD Negeri 22 sungai ambawang ( Dari Tahun 2007 - Tahun 2013 )

- SMP Bp Haruniyah Pontianak ( Dari Tahun 2013 - Tahun 2016 )

- SMK Bina Mandiri Pontianak ( Dari Tahun 2016 - 2019 )

- Sekolah Tinggi Ilmu Ekonomi Indonesia Pontianak ( Dari Tahun 2019 - Sekarang )
